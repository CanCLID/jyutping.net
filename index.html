我唔識粵拼噃，點學？

      可能需要撳多幾次，直至右下角顯示為有 icon 嘅「小狼毫輸入法」。
    3. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。

      如果見唔到有「鼠鬚管輸入法」 請去系統設定撳 鍵盤，揾 輸入方式，撳 編輯，撳右下角嘅 + 號新增鍵盤，揾「鼠鬚管輸入法」／「Squirrel」，揾到之後撳 新增，再撳過 🌐 切換去「鼠鬚管輸入法」。
    3. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。

  2. 我本身已經用緊 Rime 喇噃，噉要點樣加粵語拼音落 Rime 度？

    方法一

    直接執行上面嘅安裝檔重新安裝軟件嚟更新程式資料夾入面嘅詞庫，前提係：

    1. 從來冇執行過方法二或者方法三。因為方法二同三會將新詞庫放喺用户資料夾，呢個資料夾入面嘅檔案會優先於程式資料夾。
    2. 你冇人手改動過程式資料夾入面嘅檔案。呢啲變更喺安裝嗰陣會被覆蓋。

    方法二

    1. 用 + Space 切換去小狼毫輸入法。
    2. 喺螢幕右下角個「中」字上面撳滑鼠右鍵，揀 輸入法設定。
    3. 撳 獲取更多輸入方案…。
    4. 複製以下內容，然後喺彈出嚟個命令行界面撳滑鼠右鍵貼上： 
      cantonese emoji-cantonese CanCLID/rime-loengfan lotem/rime-octagram-data lotem/rime-octagram-data@hant
    5. 閂咗「Rime package installer」個視窗佢，去返「方案選單設定」視窗勾選「粵語拼音」，然後撳 中。
    6. 跟住揀返自己鍾意嘅配色，再撳 中 嚟完成設定。
    7. 去返個「中」字上面撳滑鼠右鍵，揀 重新部署，等個程式處理詞庫資料（需時約 20 秒，請耐心等待），之後就可以用粵語拼音打字嘞。
    8. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。

    方法三

    1. 喺螢幕左下角個 掣上面撳滑鼠右鍵，揀「終端機」（Terminal / Windows PowerShell）。
    2. 複製以下內容，然後喺彈出嚟個命令行界面撳滑鼠右鍵貼上： 
      irm https://jyutping.net/rime-cantonese-install-schema.ps1 | iex
    3. 撳 Enter 執行命令（安裝需時，請耐心等待）。
    4. 其中佢可能會彈出叫你安裝 git，要撳 允許，冇 git 裝唔到 plum。
    5. 喺螢幕右下角個「中」字上面撳滑鼠右鍵，揀 重新部署，等個程式處理詞庫資料（需時約 20 秒，請耐心等待），之後就可以用粵語拼音打字嘞。
    6. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。
    1. 複製以下內容，然後打開「終端機」（Terminal），撳 ⌘ Cmd + V 貼上： 
      curl -fsSL https://jyutping.net/rime-cantonese-install-schema | bash
    2. 撳 Enter 執行命令（安裝需時，請耐心等待）。
    3. 其中佢可能會彈出叫你安裝 git，要撳 允許，冇 git 裝唔到 plum。
    4. 打開螢幕右上角嘅輸入法選單，揀 重新部署，等個程式處理詞庫資料（需時約 10 秒，請耐心等待），之後就可以用粵語拼音打字嘞。
    5. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。
    1. 如果你已經用緊 ibus-rime 或者 fcitx5-rime，可以直接打開「終端機」（Terminal）執行以下命令（安裝需時，請耐心等待）： 
      curl -fsSL https://jyutping.net/rime-cantonese-install-schema | bash
      如果你兩樣都用緊 詞表預設會裝／更新落 ibus，如果想裝落 fcitx5，請執行以下命令： 
      curl -fsSL https://jyutping.net/rime-cantonese-install-schema | bash -s -- fcitx5
      如果你有 ibus 但未有 ibus-rime 或者有 fcitx5 但未有 fcitx5-rime，請執行 ⒈ 嘅命令。
    2. 打開輸入法選單，揀 重新部署，等個程式處理詞庫資料（需時約 10 秒，請耐心等待），之後就可以用粵語拼音打字嘞。
    3. 撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！）可以更改輸入法設定或者切換去其他輸入方式。

  5. 有冇咩工具可以幫我查粵拼？

    遇到唔識拼音嘅字可以去 jyut.net 度查，呢個係最權威嘅在線粵音字典。電腦版嘅 Rime 粵拼可以撳 ` 或者 v 或者 x 嚟用普通話拼音、倉頡、筆畫反查。啱開始用粵拼打字嗰陣可能會唔熟手，唔使擔心，打得越多就會越熟練。只要打多幾次，粵拼就會變成你最自然嘅打字方法。
