我唔識粵拼噃，點學？

  2. 我本身已經用緊 Rime 喇噃，噉要點樣加粵語拼音落 Rime 度？

    方法一

    直接執行上面嘅安裝檔重新安裝軟件，以更新程式資料夾入面嘅詞庫，前提係：

    1. 從來冇執行過方法二或者方法三。因為方法二同三會將新詞庫放喺用户資料夾，呢個資料夾入面嘅檔案會優先於程式資料夾。
    2. 你冇人手改動過程式資料夾入面嘅檔案。呢啲變更喺安裝嗰陣會被覆蓋。

    方法二

    1. 用 + Space 切換去小狼毫輸入法。
    2. 喺螢幕右下角個「中」字上面撳滑鼠右鍵，揀 輸入法設定。
    3. 撳 獲取更多輸入方案…。
    4. 然後喺彈出嚟個命令行界面，撳滑鼠右鍵貼上以下內容： 
      cantonese emoji-cantonese CanCLID/rime-loengfan lotem/rime-octagram-data lotem/rime-octagram-data@hant lotem/rime-octagram-data:customize:schema=jyut6ping3,model=hant
    5. 閂咗「Rime package installer」個視窗佢，去返「方案選單設定」視窗勾選「粵語拼音」，然後撳 中。
    6. 跟住揀返自己鍾意嘅配色，再撳 中 嚟完成設定。
    7. 去返個「中」字上面撳滑鼠右鍵，揀 重新部署，等個程式處理詞庫資料。（需時約 20 秒，請耐心等待。）
    8. 跟住撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！），就可以睇到「粵語拼音」嘞。

    方法三

    1. 喺螢幕左下角個 掣上面撳滑鼠右鍵，揀 Windows PowerShell。
    2. 然後喺彈出嚟個命令行界面，撳滑鼠右鍵貼上以下內容： 
      if (-not (Get-Command bash -ErrorAction SilentlyContinue)) { $gitBashPath = "C:\Program Files\Git\bin\bash.exe"; if (-not (Test-Path $gitBashPath)) { winget install --id Git.Git -e --source winget; } $bashCommand = $gitBashPath; } else { $bashCommand = "bash"; } & $bashCommand -c 'curl -fsSL https://git.io/rime-install | bash -s -- cantonese emoji-cantonese CanCLID/rime-loengfan custom:set:config=default,key=installed_from,value=rime-cantonese custom:clear_schema_list custom:add:schema=jyut6ping3 custom:add:schema=cangjie5 custom:add:schema=stroke custom:add:schema=luna_pinyin lotem/rime-octagram-data lotem/rime-octagram-data@hant lotem/rime-octagram-data:customize:schema=jyut6ping3,model=hant'
    3. 撳 Enter 執行命令並等待。
    4. 其中佢可能會彈出叫你安裝 git，要撳 允許，冇 git 裝唔到 plum。
    5. 喺螢幕右下角個「中」字上面撳滑鼠右鍵，揀 重新部署，等個程式處理詞庫資料。（需時約 20 秒，請耐心等待。）
    6. 跟住撳 Ctrl + `，就可以見到「粵語拼音」嘞。
    1. 打開「終端機」（Terminal），撳 ⌘ Cmd + V 貼上以下內容： 
      curl -fsSL https://git.io/rime-install | bash -s -- cantonese emoji-cantonese CanCLID/rime-loengfan custom:set:config=default,key=installed_from,value=rime-cantonese custom:clear_schema_list custom:add:schema=jyut6ping3 custom:add:schema=cangjie5 custom:add:schema=stroke custom:add:schema=luna_pinyin lotem/rime-octagram-data lotem/rime-octagram-data@hant lotem/rime-octagram-data:customize:schema=jyut6ping3,model=hant
    2. 撳 Enter 執行命令並等待。
    3. 其中佢可能會彈出叫你安裝 git，要撳 允許，冇 git 裝唔到 plum。
    4. 打開螢幕右上角嘅輸入法選單，揀 重新部署，等個程式處理詞庫資料。（需時約 10 秒，請耐心等待。）
    5. 跟住撳 Ctrl + `，就可以見到「粵語拼音」嘞。
    1. 如果你已經用緊 ibus，可以直接打開「終端機」（Terminal）執行以下命令：

  5. 有冇咩工具可以幫我查粵拼？

    遇到唔識拼音嘅字可以去 jyut.net 度查，呢個係最權威嘅在線粵音字典。電腦版嘅 Rime 粵拼可以撳 ` 或者 v 或者 x 嚟用普通話拼音、倉頡、筆畫反查。啱開始用粵拼打字嗰陣可能會唔熟手，唔使擔心，打得越多就會越熟練。只要打多幾次，粵拼就會變成你最自然嘅打字方法。
